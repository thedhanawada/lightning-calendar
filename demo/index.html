Global Timezone Support

Handle events across multiple timezones with automatic DST transitions. Store events in UTC while displaying in local time. Critical for global Salesforce organizations.

calendar.setTimezone('Asia/Tokyo');
event.getStartInTimezone('America/New_York');

Conflict Detection

Detect scheduling conflicts including time overlaps, resource conflicts, and attendee availability. Essential for resource management and scheduling.

const conflicts = calendar.checkConflicts(event);
if (conflicts.hasConflicts) {
  // Handle conflicts
}

Performance Optimized

LRU caching, lazy indexing, and batch operations. Handle thousands of events with sub-50ms query times. Optimized for large Salesforce datasets.

// Batch operations
calendar.addEvents(events);
// Query with caching
calendar.getEventsInRange(start, end);

Rich Event Model

Comprehensive event support with attendees, reminders, categories, and recurrence patterns. Map directly to Salesforce Event and Task objects.

{
  title: "Opportunity Review",
  attendees: [{email, responseStatus}],
  reminders: [{method, minutesBefore}],
  recurrence: "FREQ=WEEKLY"
}

Salesforce Native

Built specifically for Lightning Web Components. No external dependencies. Works within Locker Service restrictions.

import { Calendar } from 'c/calendarCore';

@wire(getEvents)
loadEvents({data}) {
  this.calendar.addEvents(data);
}

Zero Dependencies

Pure JavaScript implementation. No jQuery, no Moment.js, no external libraries. Reduces security audit complexity and deployment size.

// Pure JavaScript
const calendar = new Calendar();
// No external deps required

Installation

Lightning Web Component

# Copy core files to LWC directory
cp -r packages/core/src/* force-app/main/default/lwc/calendarCore/

# Deploy to Salesforce
sfdx force:source:deploy -p force-app/main/default/lwc/calendarCore

Static Resource (Visualforce)

# Create zip file
zip -r calendar-core.zip packages/core/src/*

# Upload as Static Resource in Setup > Static Resources
# Name: calendar_core

Usage

// In your LWC
import { Calendar } from 'c/calendarCore';

export default class MyCalendar extends LightningElement {
    connectedCallback() {
        this.calendar = new Calendar({
            timeZone: 'America/New_York'
        });

        // Add events
        this.calendar.addEvent({
            id: 'evt-1',
            title: 'Sales Meeting',
            start: new Date('2024-12-24T10:00:00'),
            end: new Date('2024-12-24T11:00:00'),
            timeZone: 'America/New_York'
        });
    }
}
